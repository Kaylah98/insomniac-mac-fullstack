To run the web application, install the following:
Angular CLI: 20.3.5
Angular 17
Node: 22.20.0
Package Manager: npm 10.9.3
Maven: 3.3.4
Java: Jdk 17
Spring Boot: (v3.5.7)
MongoDB Compass: To run this, go to Mongodb(.com) and download the application. Then create an application properties file in Spring Boot, use port 8080 and application name is: insomniac-mac-backend. 
OS: win32 x64
